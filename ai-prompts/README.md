# AI Prompts

Curated, tested AI prompt packs for network engineering work. Every prompt here is built around the same 4-piece anatomy (Role / Context / Constraint / Output format) and tested against real outputs before release.

## Available packs

- [**The Network Engineer's AI Prompt Pack**](prompt-engineering-network-engineers/) -- 15 prompts across config generation, troubleshooting, documentation, compliance, and code generation. v1.0 (May 2026).

## Why these are here

The companion videos for each pack walk through the structure of an AI prompt that actually produces something a network engineer would push to a device, vs. the paste-config-and-pray pattern that produces garbage. The packs are the artifact -- everything you can copy into your own AI tool tomorrow.

## Read before using

Each pack includes a safety section explaining when a prompt is safe to paste into free-tier AI and when it needs a paid tier with a no-training commitment. The short rule: if the prompt requires real configs, ACLs, hostnames, or device output to be useful, treat it as enterprise-only.
