# NetOps Toolkit

Practical incident response templates, troubleshooting scripts, and engineering playbooks for network and IT operations teams.

Built and maintained by [GTalks Tech](https://gtalkstech.com).

---

## Who This Is For

Mid-market network engineers, systems administrators, and IT operations professionals who:

- Own incidents and get blamed for everything
- Need repeatable workflows they can run under pressure
- Want copy/paste templates that actually work in the real world
- Are tired of reinventing the wheel every time something breaks

---

## What's Inside

### incident-response/templates
Communication templates for every stage of an incident:
- Initial notification to your team
- Leadership and stakeholder updates
- "It's Not the Network" evidence summaries
- Vendor and ISP escalation emails
- Post-incident summaries (RCA-lite)
- Quick status replies for "any update?" requests

Includes an AI prompt pack to speed up writing these communications.

### incident-response/checklists
Step-by-step evidence collection checklists for network triage and incident documentation.

### runbooks
Practical troubleshooting runbooks for common network and infrastructure scenarios. Each follows a Pre-check, Action, Post-check, Rollback structure.

### scripts/powershell
PowerShell scripts for network health checks, log analysis, and operational automation.

### scripts/python
Python scripts for network operations, data parsing, and workflow automation.

### ai-prompts
Curated AI prompt packs that turn raw logs, notes, and ticket data into clean incident updates, escalations, and post-incident summaries.

## How to Use This Repo

1. Browse to the folder for what you need
2. Open the relevant file (GitHub renders markdown files cleanly)
3. Click **Raw** to view plain text, or **Download** to save the file
4. Customize the templates to match your environment and organization
